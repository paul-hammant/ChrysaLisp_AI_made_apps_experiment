# Scroll

## View

```code
(Scroll flags) -> scroll
```

### :action

```code
(. scroll :action data) -> scroll
```

### :add_child

```code
(. scroll :add_child child) -> scroll
```

### :constraint

```code
(. scroll :constraint) -> (width height)
```

### :layout

```code
(. scroll :layout) -> scroll
```

### :mouse_wheel

```code
(. scroll :mouse_wheel event) -> scroll
```

### :visible

```code
(. scroll :visible descendant) -> scroll
```
